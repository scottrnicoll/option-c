# Phaser Engine Build Tracker

> **Check this file before starting work on a mechanic.** If it says "In Progress" with someone's name, pick a different one. Update this file when you start and finish a mechanic.

## How to Claim a Mechanic

1. Check that Phase 1 (Infrastructure) is **DONE** below
2. Find a mechanic with status **Not Started**
3. Change its status to **In Progress — [your name]**
4. Commit this file before you start coding
5. When all 3 options work and build passes, change status to **DONE**
